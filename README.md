# Personal Portfolio Website

A personal portfolio website built using HTML, CSS, and JavaScript to showcase my education, skills, projects, achievements, and career goals.

## Features

- Professional Portfolio Layout
- Dark Mode Toggle
- Skills Badges
- Responsive Card-Based Design
